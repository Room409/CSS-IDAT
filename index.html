Que es una especificidad

La especificidad es el nivel de importancia de una regla de estilos en relacion con otras reglas, se utiliza para determinar cual de las reglas de estilo es el que se aplica en caso de que haya conflictos o contradicciones entre ellas

Element HTML class ID In-line important
1 10 100 1000 Infinito

< p class="parrafo" id="parrafo">Hola< /p>
#parrafo{color:red;}
.parrafo{color:blue}

El ID como tiene mas valor se antepondra al class que tiene menos valor

Selectores

Son una parte fundamental para la sintaxis de CSS, se utliza para aplicar estilos visuales a un documento HTML

Selectores de etiquetas Selectores de clase Selectores de id

Tenemos por ejemplo la class que se representa con el "." y el ID con "#"

Cascada

Como bien dice el nombre Cascading Style sheets, se refiere al proceso que utliza el motor de estilos del navegador para deteminar que reglas de estilos deben aplicarser a un elemento HTML

< p> Hola < /p>
p{color:green;}
p{color:blue;}

Como todo trabaja como una cascada el ultimo elemento independientemente de lo que usemos tendra mas valor que el anterior y sera priorizado

Herencia

Se refiere a la forma en que los estilos definidos en un elemento HTML se aplican a sus elmentos secundarios.

body{color:blue;}
body h1{color:inherit;}
