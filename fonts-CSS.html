min-height

asignaremos un minimo de altura

max-width

asignaremos un maximo de ancho

Font-family:

Define el tipo de fuente que debes utilizar

Font-size:

Define el tamaño de texto que vas a aplicar

Tenemos como rem y em: rem se suma de 16 en 16 si no hay un tamaño de texto asignado, por lo contrario em y rem multiplica el tamaño de texto asginado, si el tamaño en el html asignado es de 30px, 1em o 1rem valdra 30px

font-style:

Define el estilo al texto que aplicaras

text-decoration:

Agrega una decoracion al texto que estamos añadiendo

font-weight:

Define el peso de la fuente como negra o normal

Selectores

Selector de etiquetas:

podemos llamar directamente una etiqueta de html y editarlo en css

p{color:red;}

Selector de clase

Podemos llamar clases aplicadas en las etiquetas de html a css

<p class="parrafo">hola</p> .parrafo{color:red;}

Selectores de ID

Podemos llamar la etiqueta mediante un ID que pondremos en cada una de ellas

<p id="parrafo">hola</p> #parrafo{color:red;}

Selector Universal

Podemos asignar a todos los elementos un valor default mediante un selector universal

*{font-size:40px}

Se le asignara un font-size de 40px a todos los textos

Selectores Descendientes

Podemos seleccionar una etiqueta que esta dentro de otra llamando solo al principal

<section> <p>Hola</p> </section> section p{color: red}

Selector hijo directo

Podemos llamar a los hijos directos de la etiqueta principal

<section> <h1>Hola</h1> </section> section>h1{color:red;}

Selector hermano siguiente o adyacene

Parrafo 1

Parrafo 2

Parrafo 3

div + p{color:red;}

Selector hermanos siguientes o adyacentes

Selecciona a todos los elemntos que siguen de un elemento seleccionado

<div> <p> Hola </p> </div> <p> hola1 </p> <p> Hola2 </p> div ~ p{color: red;}

Anidar selectores

Podras llamar selectores de etiquetas dentro de otras etiquetas

<div><p>Hola</p></div> div p{color:red;}
